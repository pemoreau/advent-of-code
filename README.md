# Advent Of Code

# Commands

## Rust

- `cd rust/<n>` and `cargo run`
- `cargo run --release` to run the puzzle in release mode
- `cargo test` to run unit tests

## Go

- `cd go/<n>` and `go run .`
- `go test` to run unit tests
- `go test -bench .` to run unit tests
